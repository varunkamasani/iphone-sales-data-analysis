# **Global iPhone Sales, Users & Market Trends**

## **Project Overview**
This project analyzes global iPhone sales, user growth, regional performance, and market share trends using real-world data. The goal is to derive business insights through data cleaning, exploratory analysis, and visualization.

---

## **Objectives**
- Analyze year-wise iPhone sales trends  
- Compare **USA vs Global** iPhone sales  
- Study long-term **iPhone user growth**  
- Evaluate **iOS vs Android market share**  
- Present insights using **Tableau dashboards and stories**
